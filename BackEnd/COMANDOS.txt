npm i // instala todas as dependencias

npm start //inicia o servidor

(CRIAR ARQUIVO .env NA RAIZ DO PROJETO COM A URL DO BANCO)

(PARA EFETUAR OS COMANDOS DOCKER O APP DO DOCKER PRECISA ESTAR ABERTO)

docker-compose up -d  // inicia o banco em docker

docker-compose down  // derruba o banco para manutenções

npx prisma studio  // visualiza o banco pela interface web

é preciso estar no diretorio do backend para efetuar esses comandos, do contrario irão ocorrer erros
